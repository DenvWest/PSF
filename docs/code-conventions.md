# Code conventies

- TypeScript strict, geen any
- Imports altijd via @/
- Componenten in PascalCase, files matchen component naam
- Data files in src/data/, logica in src/lib/, componenten in src/components/
- Server components default, "use client" alleen waar nodig
- Tailwind classes in JSX, geen aparte CSS files behalve globals.css
- Geen comments in code tenzij iets echt onduidelijk is
- Nederlandse UI strings, Engelse code/variabelen
