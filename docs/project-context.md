# PerfectSupplement Context

## Wat het is
Onafhankelijk supplementen-platform voor mannen 40+ met focus op 
energie, slaap, stress en hormonale veranderingen. Affiliate model.

## Doelgroep
Mannen 40-60 in Nederland, die "gewoon ouder worden" ervaren maar 
willen begrijpen wat er gebeurt en wat ze kunnen doen.

## Tech stack
- Next.js 16 App Router, TypeScript
- Tailwind CSS  
- Supabase (database)
- Hetzner VPS (hosting)
- Resend (e-mail)
- Pad alias: @/ = src/

## Huidige features
- Blog met artikelen per categorie
- Supplement-vergelijkingen (omega-3, magnesium)
- Intake flow op /intake (5 fases, 12 vragen, persoonlijk Herstelplan)
- Supabase opslag voor intake sessies, reminders en feedback
- Resend e-mail herinneringen na 30 dagen
- Cron-job.org dagelijks om 09:00
- Admin dashboard op /admin (login, Recharts grafieken, sessiedata)
- Affiliate tracking (clicks)

## Waar we heen gaan
Van content-site naar data-gedreven gezondheidsplatform. 
Intake → personalisatie → herhaalmeting → zelflerend systeem.
Later: eigen affiliate platform + white-label voor coaches.
